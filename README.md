# SevenGuard 2026 🛡️
**Ultimate Windows 7/10 Optimizer by wgofd**

Инструмент для тонкой настройки и оптимизации Windows. Позволяет снизить задержки (latency), настроить MSI Mode и провести глубокую очистку системы.

## ✨ Основные функции / Features
*   **Low Latency Optimization:** Оптимизация системы для минимального инпута-лага.
*   **MSI Mode Tool:** Настройка прерываний для видеокарт и других устройств.
*   **Deep Cleanup:** Глубокая очистка временных файлов и мусора.
*   **Registry Tweaks:** Безопасные твики реестра для повышения отзывчивости.

## 🚀 Как использовать / How to use
1. Скачайте архив из раздела [Releases](https://github.com).
2. Распакуйте и запустите `SevenOptimizer.exe` от имени администратора.
3. Выберите нужные пункты и примените настройки.

## ⚠️ Дисклеймер / Disclaimer
> Программа вносит изменения в системные настройки. Автор не несет ответственности за возможные сбои. Перед использованием рекомендуется создать точку восстановления системы.
> The program makes changes to system settings. Use at your own risk. It is recommended to create a system restore point.

## 🛠️ Разработка / Development
Проект написан на **C# (WinForms)**. Исходный код формы доступен в файле `MainForm.cs`.
